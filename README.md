# 🤖 CodeAlpha AI Chatbot

## 📌 Project Description
This is a full-stack AI chatbot application built using Python (Flask) for backend and HTML, CSS, JavaScript for frontend. The chatbot uses a TF-IDF based NLP model to understand user queries and provide relevant responses using a predefined knowledge base.

---

## 🌐 Domain
Cloud Computing & Artificial Intelligence (NLP-based Chatbot)

---

## 🚀 Features
- AI-powered chatbot using NLP (TF-IDF + Cosine Similarity)
- Real-time user interaction through web interface
- Clean and responsive UI design
- Flask REST API backend integration
- Rule-based fallback responses for unknown queries
- Lightweight and fast performance

---

## ⚙️ How to Run the Project

### 🔹 Backend Setup
1. Install dependencies:
pip install flask numpy scikit-learn

2. Run Flask server:
python app.py

---

### 🔹 Frontend Setup
1. Open `index.html` in any browser  
2. Start chatting with the AI chatbot 🤖
